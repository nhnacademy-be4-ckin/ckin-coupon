# CKIN - Coupon API Server

## ⛳️ Getting Started

1. Repository 클론

```bash
git clone https://github.com/nhnacademy-be4-ckin/ckin-coupon
```

2. ckin-coupon 폴더로 이동

```bash
cd ckin-coupon
```

3. maven project build

```bash
./mvnw clean install
```

4. spring boot 실행

```bash
./mvnw spring-boot:run
```
